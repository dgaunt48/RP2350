# VIA_6522_Tester

Program to test functionality and behaviour of a 6522 VIA IC.

    Pin 0   VGA RED
    Pin 1   VGA GREEN
    Pin 2   VGA BLUE
    Pin 3   CLOCK_LOOP_BACK
    Pin 8   VGA HORIZONTAL SYNC
    Pin 9   VGA VERTICAL SYNC
    Pin 10  CLOCK
    Pin 11  CS1
    Pin 12  #CS2
    Pin 13  Read / #Write
    Pin 14  IRQ
    Pin 15  DATA 0
    ...
    Pin 22  DATA 7
    Pin 23  ADDRESS 0
    ...
    Pin 24  ADDRESS 3
    Pin 25  #RESET
    Pin 32  PORT A
    ...
    Pin 40  PORT B
    ...
